## Why SpeedyBot?

SpeedyBot offers a complete solution for building WebEx bots with TypeScript:

### Zero dependencies

No external dependencies means a smaller footprint and fewer security concerns. SpeedyBot is self-contained and ready to use.

### Works everywhere

SpeedyBot seamlessly operates across diverse serverless and container-less platforms:

- AWS Lambda
- Cloudflare Workers
- Deno Deploy
- Azure Functions
- Traditional Node.js servers
- CI/CD pipelines

As long as you have internet connectivity, SpeedyBot functions seamlessly.

### Developer experience first

Built in TypeScript with complete type hints and autocomplete support. Features include:

- Full ESM and CommonJS support
- Live-reload on code changes during development
- Interactive documentation
- Rich IDE support with autocomplete

### Powerful abstractions

SpeedyBot shrinks away complexity and makes it a breeze to handle user input regardless of the source:

- Text messages
- File uploads
- Form submissions from SpeedyCards
- Card interactions and buttons

## Step-based architecture

SpeedyBot processes events through a sequence of steps, similar to middleware in Express. Each step can:

- Send messages or SpeedyCards to users
- Call external APIs
- Process file uploads
- Manage conversation state
- Return `$.next` to continue or `$.end` to stop

```ts
import { SpeedyBot } from "speedybot";

const Bot = new SpeedyBot();

Bot.addStep(async ($) => {
  if ($.text === "hi") {
    await $.reply(`Hi there ${$.author.name}!`);
  }
  return $.next;
});
```

## Built-in utilities

The `$` parameter provides powerful utilities:

- `$.send()` - Send messages and cards
- `$.reply()` - Reply to specific messages
- `$.card()` - Create SpeedyCards
- `$.sendFile()` - Send files to users
- `$.thread()` - Create message threads
- `$.author` - Access user information
- `$.file` - Handle file uploads
- `$.data` - Access form submissions

## SpeedyCards

SpeedyCards make it easy to build Adaptive Cards with visual embellishments and interactive forms:

```ts
const card = $.card()
  .addTitle("Welcome!")
  .addText("Choose an option below")
  .addChips(["Option 1", "Option 2", "Option 3"])
  .addTable([
    ["Feature", "Status"],
    ["Zero dependencies", "✅"],
    ["TypeScript", "✅"],
  ]);

await $.send(card);
```
