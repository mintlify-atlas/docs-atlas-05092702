---
title: SpeedyCard class
description: Builder for creating rich Adaptive Cards
---

The `SpeedyCard` class provides a fluent interface for building Adaptive Cards. All methods return `this` for method chaining.

## Constructor

```typescript
const card = new SpeedyCard()
```

Creates a new card builder instance.

## Content methods

### addTitle

Adds a title to the card.

```typescript
card.addTitle(title: string): SpeedyCard
```

<ParamField path="title" type="string" required>
  Title text
</ParamField>

### addSubtitle

Adds a subtitle to the card.

```typescript
card.addSubtitle(subTitle: string): SpeedyCard
```

<ParamField path="subTitle" type="string" required>
  Subtitle text
</ParamField>

### addText

Adds formatted text to the card.

```typescript
card.addText(text: string, config?: {
  bold?: boolean;
  size?: SizeChoices;
  align?: AlignmentChoices;
  color?: ColorChoices;
  backgroundColor?: ColorChoices;
  vertAlign?: VAlignChoices;
}): SpeedyCard
```

<ParamField path="text" type="string" required>
  Text content (supports markdown)
</ParamField>

<ParamField path="config.bold" type="boolean" optional>
  Make text bold
</ParamField>

<ParamField path="config.size" type="SizeChoices" optional>
  Text size: "Small", "Medium", "Large", "ExtraLarge"
</ParamField>

<ParamField path="config.align" type="AlignmentChoices" optional>
  Horizontal alignment: "Left", "Center", "Right"
</ParamField>

<ParamField path="config.color" type="ColorChoices" optional>
  Text color: "Default", "Dark", "Light", "Accent", "Good", "Warning", "Attention", "blue", "red", "green", "yellow"
</ParamField>

<ParamField path="config.backgroundColor" type="ColorChoices" optional>
  Background color (same options as color)
</ParamField>

<ParamField path="config.vertAlign" type="VAlignChoices" optional>
  Vertical alignment: "Top", "Center", "Bottom"
</ParamField>

**Example:**

```typescript
card.addText('Important message', {
  bold: true,
  size: 'Large',
  color: 'Attention',
  align: 'Center'
});
```

### addTable

Adds a table of key-value pairs.

```typescript
card.addTable(
  input: (string | number)[][] | { [key: string]: string | number },
  separator?: boolean
): SpeedyCard
```

<ParamField path="input" type="array | object" required>
  Array of [key, value] pairs or object with key-value pairs
</ParamField>

<ParamField path="separator" type="boolean" optional>
  Show separator line above table
</ParamField>

**Example:**

```typescript
// Array format
card.addTable([
  ['Name', 'SpeedyBot'],
  ['Version', '2.0.0'],
  ['Status', 'Active']
]);

// Object format
card.addTable({
  'Name': 'SpeedyBot',
  'Version': '2.0.0',
  'Status': 'Active'
});
```

## Survey builder

### survey

Creates a survey card with multiple questions.

```typescript
card.survey(
  questions: SurveyQuestion[],
  title?: string
): SpeedyCard
```

<ParamField path="questions" type="SurveyQuestion[]" required>
  Array of survey questions
</ParamField>

<ParamField path="title" type="string" optional>
  Survey title (default: "📝 Survey")
</ParamField>

**Example:**

```typescript
const surveyCard = new SpeedyCard().survey([
  {
    type: 'text',
    question: 'What is your name?',
    id: 'name'
  },
  {
    type: 'single-select',
    question: 'How satisfied are you?',
    choices: ['Very satisfied', 'Satisfied', 'Neutral', 'Dissatisfied'],
    id: 'satisfaction'
  },
  {
    type: 'textarea',
    question: 'Additional comments?',
    id: 'comments'
  }
], 'Customer Feedback');
```

## Card generation

### build

Builds the final card JSON.

```typescript
card.build(): Card
```

**Returns:** Adaptive Card JSON specification

### needsSubmit

Checks if the card requires a submit button.

```typescript
card.needsSubmit(): boolean
```

**Returns:** `true` if card has input fields

## Complete example

```typescript
const card = new SpeedyCard()
  .addHeader('Task Manager', {
    iconURL: '✅',
    backgroundColor: 'Accent'
  })
  .addTitle('Create New Task')
  .addText('Fill out the form below')
  .addTextInput('Task name', 'taskName')
  .addTextarea('Description', 'description')
  .addPickerDropdown([
    { title: 'High', value: 'high' },
    { title: 'Medium', value: 'medium' },
    { title: 'Low', value: 'low' }
  ], 'priority')
  .addPickerDate('Due date', 'dueDate')
  .setSubmitButtonTitle('Create Task')
  .attachData({ action: 'create_task' });

await Bot.sendTo('user@example.com', card);
```
