## Getting started with examples

All examples follow a similar pattern:

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/valgaze/speedybot
    cd speedybot/examples/<example-name>
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Configure your bot token">
    Get your bot token from [developer.webex.com/my-apps](https://developer.webex.com/my-apps) and add it to your environment.
  </Step>

  <Step title="Deploy and register webhooks">
    Follow the platform-specific deployment instructions and register your webhook URL.
  </Step>
</Steps>

## Common setup steps

### Get a bot token

1. Create a new bot at [developer.webex.com/my-apps/new/bot](https://developer.webex.com/my-apps/new/bot)
2. Copy the bot access token (you'll only see it once)
3. Save it securely - you'll need it for configuration

### Register webhooks

Most platforms require webhooks. Use the [SpeedyBot Garage](https://speedybot.js.org/garage) to:

1. Enter your bot token
2. Navigate to the Webhooks tab
3. Add your deployment URL
4. Set a webhook secret (strongly recommended)

<Warning>
Always use a webhook secret in production to validate incoming requests and prevent unauthorized access.
</Warning>
